# 题目
给你两棵二叉树的根节点 p 和 q ，编写一个函数来检验这两棵树是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

#### 示例1：
![图片](../Images/0100.jpg)
```c++
输入：p = [1,2,3], q = [1,2,3]
输出：true
```

#### 示例2：
![图片](../Images/0100.1.jpg)
```c++
输入：p = [1,2], q = [1,null,2]
输出：false
```

#### 示例3：
![图片](../Images/0100.2.jpg)
```c++
输入：p = [1,2,1], q = [1,1,2]
输出：false
```

#### 提示：

* 两棵树上的节点数目都在范围 [0, 100] 内
* $-10^4$ <= Node.val <= $10^4$
