# 题目
给定一个已排序的链表的头 head ， 删除原始链表中所有重复数字的节点，只留下不同的数字 。返回 已排序的链表 。

#### 示例1：
![图片](../Images/0083.jpg)
```c++
输入：head = [1,1,2]
输出：[1,2]
```

#### 示例2：
![图片](../Images/0083.1.jpg)
```c++
输入：head = [1,1,2,3,3]
输出：[1,2,3]
```

#### 提示：

* 链表中节点数目在范围 [0, 300] 内
* -100 <= Node.val <= 100
* 题目数据保证链表已经按升序 排列
